## What is StockExplain?
StockExplain is a web-based platform that helps Indian retail investors
understand why a stock price moved on a particular day.

Instead of showing only charts and numbers, the platform explains
price movements using simple language by analyzing market trends,
company news, trading volume, and sentiment.

## Problem Statement
Indian retail investors often panic buy or sell stocks because they do
not understand the reasons behind daily price movements.

Existing trading platforms focus on raw data and charts, which are
confusing for beginners. There is no beginner-friendly platform that
explains stock movements in plain English.

## Solution
StockExplain analyzes stock price data and relevant news to generate
clear, explainable insights that answer the question:
"Why did this stock move today?"

## Key Features (MVP)
- Search for NSE stocks
- View daily price movement
- Explanation cards for price movement
- News-based sentiment analysis
- Simple, human-readable summaries

## Tech Stack
Frontend:
- React
- Tailwind CSS

Backend:
- FastAPI (Python)

Data Sources:
- Yahoo Finance
- NewsAPI

AI & Logic:
- Rule-based explanation engine
- Sentiment analysis using NLP
